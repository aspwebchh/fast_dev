# 概述

框架是底层技术的封装，框架带来的好处是开发效率的提升， 坏处则是灵活性的丧失， 越贴近应用层的框架这两个特点体现的越明显。fast dev框架是非常贴近应用层的一个框架，能带来极大的开发效率提升，但同时也意味着不够灵活。

fast dev框架能开发三种页面类型，风别是列表页面、表单页面、数据展示页面
